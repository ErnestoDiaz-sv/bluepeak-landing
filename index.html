Security and trust

BluePeak HVAC AI is designed to support enterprise expectations for data handling and access control.

  • Data handling is designed to support least-privilege access by role.
  • Authentication uses token-based access (JWT) to limit exposure of credentials.
  • Activity is designed to be auditable with structured logs for review.
  • Encryption in transit (HTTPS) protects data between users and the app.
  • No credential sharing; each user authenticates with their own account.
